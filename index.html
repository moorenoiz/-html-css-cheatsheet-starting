HTML Cheat Sheet

Document Outline

Tag Name Description
<!DOCTYPE> Document type Declares document as html
<html> html Opening tag for html element
<head> head Page information - metadata
<body> body Page contents

Head Tags

Tag Name Description
<title> title Document title
<link> link Link to relevant resources
<meta> metadata Page metadata
<body> body Page contents

Structural Tags

Tag Name Description
<header> header Header section
<nav> nav navigation section
<main> main main content section
<section> section generic self contained section identifier
<article> article self contained composition
<aside> aside indirectly related separate content section

Basic Text Tags

Tag Name Description
<h(1-6)> Heading 1-6 Level heading
<p> paragraph element for block of text
<span> span inline section
<div> division content section divider
<a> anchor anchor or link
<ol> Ordered list Create an ordered list
<ul> Unordered list Create an unordered list
<li> list item Create a list item inside a list

Table Tags

Tag Name Description
<table> Table Table wrapper element
<thead> Table Head Rows defining table head
<tbody> Table Body Rows for table data
<th> Table Head Data Table head container
<tr> Table Row Row container
<td> Table Data Table data input
<tfoot> Table Foot Rows defining table footer

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Multimedia

Tag Name Description
<audio> Audio Embed sound content
<img> Image Embed image content
<figure> Figure Self contained content
<figcaption> Figure Caption Caption describing figure element
<video> Video Embed video content

Forms

Tag Name Description
<form> Form Document section for submitting information
<button> Button User activated interactive element
<input> Input Interactive control to take user input
